Calculate Me 🧮
A clean, responsive web-based calculator built with vanilla HTML, CSS, and JavaScript. This project focuses on a sleek UI and fundamental arithmetic operations.
🚀 Features
Core Math: Perform addition, subtraction, multiplication, and division.
Clean UI: Minimalist design using the Roboto and Ubuntu font families.
Responsive Layout: Built with a flexbox-based utility system for alignment and centering.
Interactive Design: Hover-ready buttons with smooth input handling.
🛠️ Built With
HTML5: Semantic structure for the calculator interface.
CSS3: Custom styling including a dedicated utility file for flexbox and alignment.
JavaScript: Logic for handling button clicks and evaluating expressions.
📂 Project Structure
index.html: The main structure of the calculator.
style.css: Primary styling, including font imports and button layouts.
utils.css: Utility classes for centering, flexbox, and margins.
script.js: Logic for the calculation engine and event listeners.
⚙️ How It Works
The calculator uses an event-driven approach where:
Input: Users click buttons to build a mathematical string.
Calculation: Upon clicking =, the string is evaluated and the result is displayed in the input field.
Clear: Clicking C resets the current string and display.
